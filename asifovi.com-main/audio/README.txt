BACKGROUND MUSIC INSTRUCTIONS

To add background music to your portfolio website:

1. Download a royalty-free ambient music file from one of these sources:
   - Pixabay: https://pixabay.com/music/search/ambient/
   - Free Music Archive: https://freemusicarchive.org/genre/Ambient/
   - Bensound: https://www.bensound.com/royalty-free-music/ambient

2. Rename your downloaded file to one of these names:
   - calm-ambient.mp3 (preferred)
   - background-music.mp3 (alternative)

3. Place the file in this directory (audio/)

4. Refresh your website and the music player should now work!

TROUBLESHOOTING:
- Make sure the file is in MP3 format
- Keep the file size under 5MB for better performance
- If the player still doesn't work, check the browser console for errors
- Some browsers require user interaction before playing audio
